Sobre mim :

Sou um desenvolvedor front-end com sólido conhecimento em tecnologias essenciais para a criação de interfaces web, incluindo HTML, CSS e JavaScript. Com habilidades em design responsivo, estou comprometido em criar experiências de usuário intuitivas e atraentes.

Principais projetos :

  • Projeto I-phone.


    Projeto em referência ao lançamento I-phone 14, onde é possível selecionar sua cor preferida.
    Tecnologias utilizadas: Html, CSS e JavaScript.

  • Projeto Santander.


    Projeto em referência a página do banco Santander, trata-se de um projeto criativo onde foram aprimoradas técnicas no CSS, como exemplo as pseudo-classes.
    Tecnologias utilizadas: Html e CSS.

  • Projeto Conversor de Moedas.


    Trata-se de um sistema de conversão de moedas onde é possível converter de Real Brasileiro para Dólar Americano, Euro, Libras e BitCoin.
    Tecnologias utilizadas: Html, CSS e JavaScript.

  • Easy Shopping Via Mobile.


    Projeto de uma página de uma loja de compras, trata-se de um projeto criativo onde foram aprimoradas técnicas no CSS, aplicando a responsividade tornando o projeto mais versátil.
    Tecnologias utilizadas: Html e CSS.
